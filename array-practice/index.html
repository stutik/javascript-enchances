Here some are the example of array methods:-

1.Push method:-

push method is a method to push element in the last element of the array.

const addFruits = ["apple", "banana", "kiwi"]; addFruits.push("orange"); console.log(addFruits); result:- ['apple', 'banana', 'kiwi', 'orange']

2.Unshift method:-

unshift method is a method to push element in the first element of the array.

const addFruits = ["apple", "banana", "kiwi"]; addFruits.push("orange"); console.log(addFruits); result:- ['orange','apple', 'banana', 'kiwi']

3.pop method:-

pop method remvoe the last element in the array

const deleteFruits = ["apple", "banana", "kiwi"]; deleteFruits.pop(); console.log(deleteFruits); ['apple', 'banana']
